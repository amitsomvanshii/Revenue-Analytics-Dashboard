# Revenue-Analytics-Dashboard
## Revenue Analytics Dashboard

## Short Description and Purpose
This project presents a Revenue Analytics Dashboard built using Power BI to analyze sales performance, profitability, and customer behavior. It combines multiple datasets to generate actionable insights that help businesses understand revenue trends, identify profit-driving categories, and improve decision-making.

## Tech Stack
Power BI Desktop – Dashboard creation & visualization
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated measures & KPIs
Data Modelling – Relationship building between datasets
File Format – Revenue Analytics Report.pbix (Power BI project file)

## Data Sources
The project uses two datasets:
1. details.csv
Order ID
Amount
Profit
Quantity
Category
Sub-Category
Payment Mode

2. orders.csv
Order ID
Order Date
Customer Name
State
City

## Features and Highlights
# Business Problem
Businesses often struggle to:
Track revenue and profit trends effectively
Identify high-performing products and regions
Understand customer purchasing patterns

# Goal of the Dashboard
Provide a clear view of revenue and profit performance
Identify top-performing categories and sub-categories
Analyze regional and customer-level insights
Enable data-driven business decisions

# Walkthrough of Key Visuals
KPIs Cards – Total Revenue, Profit, and Quantity
Category-wise Analysis – Revenue and profit by category & sub-category
Geographical Insights – State and city-wise performance
Time-based Trends – Sales trends over time (monthly/yearly)
Payment Mode Analysis – Distribution of transactions by payment type

# Business Impact & Insights
Identifies high-profit and loss-making segments
Helps optimize product strategy and pricing decisions
Reveals top-performing regions and customer segments
Supports strategic planning and revenue growth
